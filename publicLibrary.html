Public Library

There is a book introduces all details about developing these applications, you can get it now!

  1. Public Library with Minimal Effort

    building a front-end web application with minimal effort using AngularJS. The app supports the four standard data management operations (Create/Read/Update/Delete).

  2. Public Library with Constraint Validation

    perfoming constraint validation both in the controller part of the app and in the user interface built with AngularJS.

  3. Public Library with Unidirectional Association

    It extends the example app of part 2 by adding code for handling the unidirectional functional (many-to-one) association between Book and Publisher, and the unidirectional non-functional (many-to-many) association between Book and Author.

  4. Public Library with Bidirectional Association

    A bidirectional association is an association that is represented as a pair of mutually inverse reference properties.

  5. Public Library with Subtyping

    building a web app that manages subtype (inheritance) relationships between object types. Whenever an app has to manage the data of a larger number of object types, there may be various subtype (inheritance) relationships between some of the object types.
